# openAI Project

Bu proje üç ana bileşenden oluşur:
- **backend**: Sunucu tarafı kodları.
- **frontend**: Kullanıcı arayüzü (Next.js).
- **contracts**: Hardhat ile geliştirilen akıllı sözleşmeler.

## Kurulum

1. Her dizine girerek bağımlılıkları yükleyin:
   ```bash
   cd backend && npm install
   cd ../frontend && npm install
   cd ../contracts && npm install
   ```

2. `.env` dosyalarınızı ayarlayın.

3. Uygulamayı başlatın.
